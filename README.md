# The salon

## Live demo:

[https://the-salon.netlify.app/](https://the-salon.netlify.app/)

## Installation

### Clone the repository

```bash
git clone https://github.com/lulu70/the-salon.git
```

### Install dependencies

```bash
npm install --save
```

### Start dev server

```bash
npm start
```
